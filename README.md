## üîß **Quick Start**

### Prerequisites
- Rust 1.70+
- Node.js 18+
- Python 3.8+

### Build Core
```bash
cd core/qnet-core
cargo build --release
```

### Start Node
```bash
cd infrastructure/qnet-node
python src/node.py
```

### Launch Explorer
```bash
cd applications/qnet-explorer
npm install
npm run dev
```
